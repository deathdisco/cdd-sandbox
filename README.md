CDD RPC Web Sandbox
===================
[![License](https://img.shields.io/badge/license-Apache--2.0%20OR%20MIT-blue.svg)](https://opensource.org/licenses/Apache-2.0)

## Development

```bash:
npm install
npm start
```

### Or using NixOS...
```bash
nix-shell --command "npm start"
```

## Installation (Production)

```bash:
npm run build
```
